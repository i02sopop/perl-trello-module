# Trello

This perl module uses the Trello API to manage the boards. It depends on
IO::Socket::SSL, Moose, Role::REST::Client and URI::Escape.

## INSTALLATION

To install this module, run the following commands:

``` shell
perl Build.PL
./Build installdeps
./Build
./Build test
./Build install
```
